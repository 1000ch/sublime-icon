# sublime-icon

Utility for Sublime Text to set or replace its icon.

## Install

```bash
$ npm install -g sublime-icon
```

## Usage

```bash
$ sublime-icon --help
```

## Icons

- [t32k/Sublime-Text-2-Icon](https://github.com/t32k/Sublime-Text-2-Icon)
